# Mineflayer Bot Builder

A visual, node-based interface for creating [PrismarineJS/mineflayer](https://github.com/PrismarineJS/mineflayer) bots without coding. The interface uses [litegraph.js](https://github.com/jagenjo/litegraph.js) to provide a simple drag-and-drop experience for building Minecraft bots.

## Features

- Visual node editor for creating Minecraft bots
- Connect to Minecraft servers with customizable settings
- Support for various Mineflayer functions and events
- Pathfinding capabilities via mineflayer-pathfinder
- Save and load your bot configurations

## Installation

```bash
# Clone the repository
git clone https://github.com/yunusemregul/mineflayer-bot-builder.git

# Navigate to project directory
cd mineflayer-bot-builder

# Install dependencies
npm install

# Start the application
npm start
```

## Usage

1. Use the node editor to create your bot logic by connecting nodes
2. Configure your bot connection settings (host, port, username, etc.)
3. Click "Run" to start your bot
4. Save your configuration for future use

## Available Nodes

- **Bot Creation**: Create and configure bots
- **Events**: Handle Minecraft events (chat, spawn, etc.)
- **Actions**: Perform various bot actions (movement, chat, etc.)
- **Pathfinding**: Navigate the Minecraft world
- **Logic**: Conditional statements and flow control
